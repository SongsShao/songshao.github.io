Hexo and github.io

2019-08-01

​ Welcome to SongShao Notes.Today I will tell you how to use Hexo and github to create your own static blog.

​ First we need to install several software, npm and git, npm is responsible for downloading the local resource files we need, git is responsible for connecting the github project branch to upload and download files.The node.js here already contains npm so there is no need to install it.

Configuring SSH key

​ Why should I configure this? Because you must have your github permission to submit the code, but it is too insecure to use the username and password directly, so we use ssh key to solve the connection problem between the local and the server.

​ If prompted: No such file or directory indicates that you are using git for the first time.

​ Then enter 3 times in succession, and finally generate a file in the user directory, open the user directory, find .ssh\id_rsa.pub file,Notepad opens and copies the contents, open your github home page, enter personal settings-> SSH and GPG keys -> New SSH key：

​ Fill in the key just copied in the location of the key.

Test Connection
